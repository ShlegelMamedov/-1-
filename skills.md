пр3 вопросы 
git config --global user.name/email — настраивают имя и почту автора коммитов. Эти данные отображаются в истории изменений на GitHub.
Проверка установки Git: в терминале выполните git --version — если показана версия (например, git version 2.45.0), всё установлено верно.
Просмотр истории:
В GitHub Desktop: вкладка History
В терминале: команда git log
Ветка (branch) — изолированная копия кода для разработки новой функции без риска сломать основную версию. Позволяет работать над несколькими задачами параллельно.
Файл .gitignore — указывает Git, какие файлы/папки игнорировать (не отслеживать). Используется для исключения временных файлов, логинов, скомпилированного кода, чувствительных данных.
